# Desktop Autoclicker - Download Instructions
Download of the autoclicker is straight forward, but can be tricky to use due your computer's protections. Here is how to navigate this:




**• 1: Download:**\
Download the application from the "Autoclicker Download" folder in this repo.
\
**• 2: Unzip:**\
Unzip the file. You might need a file unziper application if you do not already have one.
\
**• 3: Localize the Executable:**\
To find the executable file, follow *yume_clicker_v1 > dist > autoclicker*
\
**• 4: Run:**\
The autoclicker might trigger your computer's security system, requiring extra permissions. In the pop up window, find a "Run Anyway" option.
\
**OPTIONAL: Create a shortcut to the executable**\
Right click your desktop, head into *New > Shortcut*. Find the executable and set it as the destination.
